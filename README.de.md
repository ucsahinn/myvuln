MyVuln ist eine Threat-Intelligence-Arbeitsfläche für Security-Teams. Sie bündelt CVE-Tracking, URL-Analyse, Intel Feed, API-Integrationen, Alerts und Reporting.

## Wichtig

Dieses Repository ist keine Source-Code-Ablage. Es ist die öffentliche, sichere Produkt- und Supportfläche für MyVuln. Private Source, Supabase-Migrationen, operative SQLs, Tenant-Daten, Logs und Secrets gehören nicht hierher.
