MyVuln é uma área de threat intelligence para equipes de segurança. Ela reúne acompanhamento de CVEs, análise de URLs, Intel Feed, integrações de API, alertas e relatórios.

## Importante

Este repositório não contém o código-fonte. Ele é a superfície pública e segura de produto e suporte para o MyVuln. Código privado, migrações Supabase, SQL operacional, dados de tenants, logs e segredos não pertencem a este repositório.
